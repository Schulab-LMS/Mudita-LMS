# SchuLab Character System — Lumo

One main STEAM guide, **Lumo**, appears across the platform with six age-band variants and topic costumes. The Character Consistency Agent maintains a locked prompt-template library so every generated image/video stays on-model.

## Core identity (invariant across all variants)

Same face DNA · same color family · same logo mark · same friendly expression — the "SchuLab visual DNA." Personality: curious, encouraging, funny-not-silly, simplifies hard ideas, loves missions and challenges.

## Age variants

| Variant | Band | Style |
|---|---|---|
| **Lumo Mini** | 3–5 | very cute, soft shapes, big eyes, bright colors, slow speech, parent-guided tone |
| **Lumo Junior** | 5–7 | playful, storytelling-focused, simple words, props (stars, blocks, paint, story cards) |
| **Lumo Creator** | 8–10 | energetic, game-like, missions/badges, Scratch/coding/project visuals |
| **Lumo Explorer** | 11–13 | curious, science-focused, problem-solving language, gadgets/sim/space/robotics dashboard |
| **Lumo Innovator** | 14–16 | more mature, tech-focused, real-world projects, laptops/dashboards/AI tools/prototypes |
| **Lumo Mentor** | 17–18 | professional-but-friendly, portfolio/leadership/product thinking, startup/AI/data/capstone visuals |

## Topic costumes

Space → astronaut headset/badge · AI → small robot sidekick · Coding → code blocks/laptop · Science → sim screen / lab-style digital board · Web/App → UI wireframes · Game design → game-world style · Earth/Environment → globe / satellite map.

## Consistency rules

Lumo must remain recognizable across all versions: same core face identity, same color family, same logo mark, same friendly expression, same SchuLab visual DNA.

## Deliverable

A `character-system.md` plus a locked **prompt-template library** (per variant × topic) and a reference image set (style anchors) so generation stays on-model. Stored under `docs/curriculum-production/characters/`. The Character Consistency Agent selects the variant from this canonical set — it never invents a new character.
