# Example — PhET (ages 11–13), video + character only (Mode 3)

**Command:** "Use the SchuLab curriculum generator skill to create only the video
script and character prompts for a PhET simulation lesson about forces and
motion for ages 11–13."

## Process (Mode 3 — video script only)
1. **Bind to catalog:** an existing ages-11–13 science course. Lumo variant =
   **Lumo Explorer** (science costume: simulation screen / lab board). Video 4–7 min.
2. **Source:** PhET is a **Simulation** source (CC BY). The lesson is
   simulation-driven — **embed/link** the PhET sim (resource type `link`) and
   **attribute** PhET; never copy PhET's own text/images.
3. Output **only** `video-script.md` + `character-prompts.md`.

## video-script.md (excerpt)
- Target: ages 11–13 · Lumo Explorer · ~5 min · voiceover: curious, problem-solving.
- **Scene — predict before you test:** Lumo Explorer at a sim screen. Narration:
  "Predict first — what happens to the box if we push twice as hard?" On-screen:
  "PREDICT → then TEST in the simulation." Interaction pause: open the PhET
  Forces & Motion sim and try it. Quiz popup: "More force →?".
- CTA: "Run your own experiment in the simulation and record what you find."

## character-prompts.md (excerpt)
- Invariant DNA restated (face/color/logo/expression).
- Main prompt: *Lumo Explorer (ages 11–13 variant) beside a glowing physics
  simulation screen showing a box on a ramp, friendly and curious, SchuLab
  colors, on-model.*
- Thumbnail: Lumo Explorer pointing at a force arrow on the sim.

## Guardrails
- No unsupported claims — everything ties to what the PhET sim actually shows.
- CC BY attribution to PhET present; `review-checklist.json` →
  `noCopyrightCopying`, `videoMatchesLesson`, `characterConsistent` to be human-verified.
- `approvedForPublishing: false`.
