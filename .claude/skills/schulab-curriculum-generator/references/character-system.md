# SchuLab character system — Lumo

One main STEAM guide, **Lumo**, appears across the platform with six age-band
variants and topic costumes. Lumo must stay recognizable everywhere.

**Lumo** — friendly STEAM guide. Personality: curious, encouraging, clear,
friendly, funny-but-not-silly, explains hard ideas simply, loves missions and
challenges.

## Age variants (select by the lesson's age band)
| Variant | Band | Style |
|---|---|---|
| **Lumo Mini** | 3–5 | cute, soft, simple; big expressions; very gentle tone; parent-guided |
| **Lumo Junior** | 5–7 | playful, storytelling-focused; props (stars, blocks, paint, story cards) |
| **Lumo Creator** | 8–10 | energetic, mission-based; badges, Scratch/coding/project visuals |
| **Lumo Explorer** | 11–13 | science & problem-solving; gadgets, simulations, space tools, robotics dashboard |
| **Lumo Innovator** | 14–16 | more mature, tech-focused; laptops, code, dashboards, AI tools, prototypes |
| **Lumo Mentor** | 17–18 | professional-but-friendly; portfolio, leadership, product thinking, capstone |

## Topic variations (costume/accessory)
- Space → space badge, astronaut headset, star map
- AI → small robot assistant
- Coding → code blocks or laptop
- Science → simulation screen or digital lab board
- Web/App → UI wireframes
- Game design → game-world visuals
- Earth/environment → globe or satellite map

## Consistency rules (invariant across all versions)
- Same core face identity.
- Same color family.
- Same SchuLab visual identity / logo mark.
- Same friendly expression.
- Age/topic variations are allowed, but **Lumo must remain recognizable**.

When writing `character-prompts.md`, every prompt must restate this DNA so
generated images stay on-model (see the template).
