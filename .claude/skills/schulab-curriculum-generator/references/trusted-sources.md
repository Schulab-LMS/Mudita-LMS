# Trusted source library

Use ONLY these approved sources. No fabricated sources, providers, or URLs. This
mirrors the platform's `ReferenceSource` library (`prisma/catalog/reference-sources.data.ts`).

## 6-category classification (the `sourceType`)
| Category | Grounds curriculum? |
|---|---|
| **Core curriculum source** | ✅ yes |
| **Activity source** | ✅ yes |
| **Simulation source** | ✅ yes (simulator-first) |
| **Enrichment source** | ❌ enrichment/curiosity only |
| **Marketplace/discovery inspiration** | ❌ inspiration only, never grounding |
| **Historical reference only** | ❌ inspiration only |

Only Core curriculum, Activity, and Simulation sources may ground a lesson.
Enrichment, Marketplace, and Historical sources are never copied or used to
generate factual content.

## Special rules
- **Google CS First / Experience CS** — historical inspiration only, not an active source.
- **Outschool** — marketplace/course-discovery inspiration only; never for curriculum structure or content.
- **National Geographic Kids** and **Smithsonian** — mainly enrichment; do not copy their proprietary media.
- **micro:bit** and **Raspberry Pi** — simulator-first / optional. Never require physical hardware.
- **LEGO, Arduino, robotics kits, physical science kits, or shipped materials must NOT be required** for the main SchuLab e-learning experience. Activities are fully digital.
